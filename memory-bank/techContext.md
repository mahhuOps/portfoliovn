# Tech Context

**Technologies Used:**

*   Next.js
*   React
*   TypeScript
*   Tailwind CSS
*   Firebase
*   Radix UI
*   Lucide React

**Development Setup:**

*   The project requires Node.js and npm to be installed.
*   The project can be run using the `npm run dev` command.

**Technical Constraints:**

*   The project must be deployed on Vercel.
*   The project must be secure and performant.

**Dependencies:**

*   The project depends on the following npm packages: `@hookform/resolvers`, `@radix-ui/react-accordion`, etc.
